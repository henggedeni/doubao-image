## 排查 Missing credentials 报错
- 报错：`Missing credentials. Please pass an apiKey... or set the OPENAI_API_KEY`
- 原因：`.env` 中 `KIMI_API_KEY=sk-kimi-xxx` 是占位符，未填入真实 Key
- ChatOpenAI 的 openAIApiKey 传入假值时抛出此错，提示 OPENAI_API_KEY 只是默认建议
- 修复：需在 .env 填入 Kimi 开放平台真实 API Key
- 确认：kimi-k2.6 模型支持视觉识别，base URL https://api.moonshot.cn/v1 正确
- .env 中 DeepSeek 相关配置已无用，可删除
